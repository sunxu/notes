 Linux 上挂载exFAT格式的移动硬盘
分类： Linux 2012-01-21 15:14 790人阅读 评论(1) 收藏 举报
由于之前下了一个文件大小大于4G，原来的移动硬盘放不下单个这么大的文件，所以把移动硬盘格式化成exFAT格式的，但是这样却造成了使用Linux系统就挂载不上这个移动硬盘，因为Linux不能识别exFAT文件系统，这让我非常郁闷，于是上网找了下，发现很多同学都遇到了这种问题，也有大牛放出了解决的方法，于是按照网上的方法来搞一下，一开始很悲剧的失败了，因为网上的解决方法是在ubuntu上的，但我的是fedora，有一个软件包死活没装上，把我搞了哥焦头烂额，最后还是在google大哥的帮助下解决了问题，真是不容易啊，下面写一下我的解决方法：


1.安装以下软件包（用su身份）

#yum install subversion scons libfuse-devel gcc

（其中libfuse-devel包在ubuntu中是libfuse-dev包，就是这个不同让我浪费了不少时间）

$svn co http://exfat.googlecode.com/svn/trunk/ exfat-read-only

$cd exfat-read-only

$scons

#scons install

#cd ..

#rm –rf exfat-read-only
